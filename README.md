# Trivia game

Opdracht: https://www.quad.team/assignment

# Aannames

Aannames op basis van de tekst van de Trivia game:
- Met Open Trivia Database wordt https://opentdb.com/ bedoeld.
- De antwoorden worden pas gechecked aan het eind, want de POST endpoint verwacht een lijst.

Extra aannames om de opdracht interessant te maken:
- De klant wil weten hoeveel vragen gebruikers goed hebben.
- Gebruikers registreren zichzelf, waarbij het emailadres wordt opgeslagen.
- De klant kan beveiligd inloggen en emailadressen koppelen aan het aantal goede antwoorden.
